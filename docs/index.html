Movie Recommendation System

This GitHub Pages site is a static landing page for the project. The full recommendation app is built with Streamlit and must run on a Python host.

View Source on GitHub Deploy Streamlit App

What works on Pages

Documentation, screenshots, and static HTML/CSS/JS.

What does not run on Pages

Python backend execution such as streamlit run app.py.

Production path

Host the Streamlit app on Streamlit Community Cloud, then link it here.

After GitHub Pages is enabled in repository settings, this page will be published automatically from the docs artifact via GitHub Actions.
