🎬 Movie Recommendation System

A full-stack recommendation engine built on the MovieLens dataset, combining SVD Collaborative Filtering and TF-IDF Content-Based Filtering with an interactive Streamlit web app.

View on GitHub Quick Start

📸 App Preview

✨ Features

📥

Data Loading

Auto-downloads the MovieLens Small dataset; falls back to synthetic data when offline.

🧹

Data Cleaning

Deduplication, invalid-rating removal, title and genre parsing.

📊

EDA Dashboard

Rating distributions, genre popularity, activity over time and top-rated movies.

🤝

Collaborative Filtering

SVD matrix factorisation via scikit-surprise for personalised picks.

🎭

Content-Based Filtering

TF-IDF on genres & user tags with cosine-similarity ranking.

🔀

Hybrid Mode

Weighted blend of CF and CBF scores for more robust recommendations.

📏

Evaluation

RMSE, MAE, Precision@K and Recall@K metrics on a held-out test set.

🌐

Streamlit App

Interactive UI with 6 recommendation modes — no frontend knowledge needed.

🌐 App Modes

ModeDescription
🔍 Search & BrowseFull-text title search with genre filter
👤 Recommend for User (CF)Personalised recommendations using SVD
🎭 Similar Movies (CBF)Find movies similar to a title via TF-IDF
❤️ Based on Liked Movies (CBF)Recommendations from a list of liked titles
🔀 Hybrid RecommendationsWeighted blend of CF + CBF
📊 EDA DashboardInteractive exploratory data analysis

🚀 Quick Start

  1. Clone the repository

    git clone https://github.com/kilionrotich/Movie-review-system.git
cd Movie-review-system

  2. Install dependencies

    pip install -r requirements.txt

  3. Train the models

    python train.py

  4. Launch the Streamlit app

    streamlit run app.py

    Open http://localhost:8501 in your browser.
