Movie Recommendation System

A full-stack recommendation engine combining SVD Collaborative Filtering and TF-IDF Content-Based Filtering on the MovieLens dataset, served via Streamlit.

Streamlit App

Deploy the interactive app on Streamlit Community Cloud by pointing it at app.py on the main branch.

Deployment Files

  • requirements.txt – pinned dependencies
  • runtime.txt – Python 3.12
  • .streamlit/config.toml – server config
  • .github/workflows/ci.yml – CI check

Quick Deploy to Streamlit Cloud

  1. Go to share.streamlit.io
  2. Sign in with GitHub
  3. New app → kilionrotich/Movie-review-system
  4. Branch: main, File: app.py
  5. Click Deploy
GitHub Pages hosts static content only. The full Streamlit app requires a Python host such as Streamlit Community Cloud.
